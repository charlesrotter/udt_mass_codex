## ★ Main structure

### 1. Exact all-\(z\) ruler (any seat, looking outward on L)

\[
\boxed{H_{\mathrm{loc}}\,\ell_{\mathrm{proper}}=\frac{z}{1+z}}
\]

Equivalent forms:
- \(\ell / L_{\mathrm{remain}} = z/(1+z)\) with \(L_{\mathrm{remain}}=1/H_{\mathrm{loc}}\)
- \(z = (H\ell)/(1-H\ell)\) for \(H\ell<1\)
- Wall when \(H\ell\to 1\) (proper room used up)

Numeric identity holds for all tested seats and depths (including \(z=9\) at 90% of room used).

### 2. Universal distance–redshift shape (only scale is seat’s \(H\))

For **every** seat (center or off-center), small-angle radial line of sight:

\[
\boxed{d_L=\frac{z(z+2)}{2\,H_{\mathrm{loc}}},\qquad
d_A=\frac{1-1/(1+z)^2}{2\,H_{\mathrm{loc}}}}
\]

Center recovers \(d_L=X\,z(z+2)\) because \(H_{\mathrm{loc}}(0)=1/(2X)\).

**Lay:** Changing seat only changes the ruler scale (local Hubble / room-to-wall). The *shape* of dimming and size vs redshift is the same map everywhere. That is pure kaleidoscope.

### 3. Coasting cosmography at every seat (\(q_0=0\))

\[
\frac{d_L\,H_{\mathrm{loc}}}{z}=1+\frac{z}{2}\quad\text{(exact, all \(z\), all seats).}
\]

Standard low-\(z\) reading: \(d_L=(z/H)[1+(1-q_0)z/2+\cdots]\) ⇒ **\(q_0=0\)** at every seat — not a center speciality.

### 5. Mild \(G(\theta)\): same law, anisotropic sky

Along each ray with constant \(G(\theta)\) (separable \(A=H(r)+G(\theta)\)):
- \(H_{\mathrm{loc}}(\theta)\times L_{\mathrm{remain}}(\theta)=1\) still

Sky anisotropy without new forces — residual direction only.

### 6. Two-eye reciprocity (radial small-angle)

\[
\frac{D_A(o\to s)}{D_A(s\to o)}=\frac{1}{1+z}
\]

Redshift is symmetric (\(z_{os}=z_{so}\)); angular-diameter distance is not — each seat’s residual rescales angles.
