# Strongest current global reciprocal-closure skeleton

## Ruling

The current foundation supports a coherent **working skeleton**, but not a closed global object or a
global variation principle.

The strongest statement that can be assembled without silently adding a join is:

> A realized UDT universe is sought as a finite whole-solution domain whose local pre-scale metric
> data contain a determinant-one reciprocal clock/ruler direction inside a Common-Scale-Neutral
> class. A scoped static spatial seal fixes `phi=0` while leaving its normal derivative free.
> Positive common rescaling preserves the local causal cones. A realized matter-bearing solution
> must satisfy the owner-stated global bootstrap admissibility condition.

Every sentence above is supported, with its status stamp, by `CLAUSE_LEDGER.tsv`. The sentences do
**not** yet say how the local metric data glue into one physical finite metric, what the complete seal
does to every coframe sector, which whole-object variations are legal, how bootstrap acts off shell,
or how a physical scale and matter source emerge.

- `J01`: the physical full-coframe slot/readout map;

What has been learned is that they are the same *layer* of the problem: defining the whole geometric
object before asking an action or bootstrap to select a solution on it.

## Why local equation chasing can mislead

The latest transverse-coframe audit supplied a concrete warning. Freezing unresolved angular
cap/period data produced an apparent radial shear equation that rejected an exact Bach-flat Einstein
branch. The selector was an artifact of the restricted variation domain. This shows that local
Euler-equation exploration cannot safely decide the global branch until the complete variation space
is known.

The appropriate next metric-led target is therefore not another guessed local term. It is a global
cocycle/gluing and boundary-tangent theorem for the existing reciprocal coframe across a finite cell.
If the metric does not force that theorem, the negative result should be banked rather than repaired
with a mechanism.

The skeleton is therefore `WORKING`, not `DERIVED`, and it is not canon.
