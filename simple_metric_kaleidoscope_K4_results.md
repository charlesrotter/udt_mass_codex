## ★ What emerges

### 1. Redshift is path-blind; angles are not

For stationary observers on the static residual metric, Killing time ⇒  
\(1+z=\sqrt{A_o/A_s}\) **independent of the light path**.

Angular place on the sky and angular size **do** depend on the null geodesic (impact parameter).  
Kaleidoscope splits: **clock residual is local-to-local; sky map is path geometry.**

### 2. Off-axis sticks look bigger (residual lensing geometry)

Same seat \(r_o\), same source depth \(r_s\), fixed proper stick, vary sky angle \(\gamma\) off the radial line:

| \(\gamma\) | \(D_A/D_A^{\mathrm{rad}}\) (example \(r_o=0.2,\,r_s=0.75\)) |
|------------|---------------------------------------------------------------|
| \(0^\circ\) | \(1.00\) |
| \(20^\circ\) | \(\sim 0.85\) |
| \(35^\circ\) | \(\sim 0.58\) |
| \(50^\circ\) | \(\sim 0.29\) |

Smaller \(D_A\) ⇒ **larger angular size**. Off-axis residual geometry acts like a wide-angle amplifier — no imported lens mass; pure place + null rays.  
(Numeric: two neighboring geodesics; radial limit matches analytic \(D_A=(r_s-r_o)/\sqrt{A_o}\).)

### 3. Multi-probe = one scale-free map

With room-to-wall \(L_{\mathrm{remain}}=1/H_{\mathrm{loc}}\):

\[
\frac{d_L}{L_{\mathrm{remain}}}=\frac{z(z+2)}{2},\qquad
\frac{d_A}{L_{\mathrm{remain}}}=\frac12\Bigl(1-\frac{1}{(1+z)^2}\Bigr),\qquad
\frac{\ell_{\mathrm{proper}}}{L_{\mathrm{remain}}}=\frac{z}{1+z}.
\]

Etherington holds identically. Absolute distances are set only by the seat’s \(H_{\mathrm{loc}}\) (or room).  
SNe / size / “Hubble length fraction” are **three readouts of one residual function of \(z\)**.

### 4. Mild sky residual → magnitude anisotropy at fixed shell

Center seat, \(A=1-r/X+\varepsilon P_2(\cos\theta)\), fixed areal \(r_s=0.7X\), \(\varepsilon=0.05\):

- \(z\) contrast \(\max/\min\sim 1.21\)
- \(d_L\) contrast \(\sim 1.18\)
- peak-to-peak \(\Delta m \sim 0.36\,\mathrm{mag}\)

Direction-dependent dimming on a fixed shell — residual sky only.

### 5. Inward look: blueshift

Seat deep, source shallower (\(A_s>A_o\)):

\[
z=\sqrt{A_o/A_s}-1<0
\]

**Looking toward the center is looking toward lower residual depth → blueshift.**  
The kaleidoscope is two-way: outward reddens and dims; inward blueshifts. Not an expanding-space story — residual between seats.
