# Independent-verifier shear-mode correction

The first independent invocation passed the background action-density comparison, both full-Bach
zero controls, the conformal restricted-shear control, every twist/even mixed block, the nonzero pure
twist diagnostic, and the constant-connection zero mode. It stopped only because the Einstein
restricted-shear comparison used different variations on its two sides.

The exact expected value `-32 K^2 F/3` is for a constant relative rescaling of the two angular legs.
The verifier had inserted the `r`-dependent test mode `1+r/9+r^2/13`; its pointwise action derivative
therefore contained derivatives of that test mode and could not equal the constant-mode expression.

The verifier is corrected to use the constant mode `1` for that registered variation-domain
witness and the mixed-block checks. Reflection predicts the latter vanish for every even test mode,
so this does not weaken their gate. No tolerance, metric profile, curvature sign, Bach equation,
action formula, or outcome was changed. The failed pre-bank invocation supplies no scientific
result.
