1. `REFERENCE.md` for the whole scientific structure in one place;
2. `CLAIM_DEPENDENCY_LEDGER.tsv` for exact premise and evidence dependencies;
3. `OPEN_JOIN_LEDGER.tsv` for every unresolved arrow and prohibited shortcut;

5. `EXTERNAL_REVIEW_PACKET.md` for the cold-review firewall and role assignments.

atlas is a `WORKING` question only. External reviews are pending at this draft checkpoint.
