# Completeness map

| Axis | Covered | Not covered |
|---|---|---|
| Metric family | Arbitrary `C^2 A(r)>0` static spherical areal metric | Nonspherical or time-live metric |
| Frame type | Exact finite-`phi` rank/signature and orthonormal coframe | Material observers, acceleration law, micro theorem |
| Causal readout | Radial coordinate and locally normalized metric-null slopes | Universal signal/carrier coupling |
| Angular interaction | Complete static spherical curvature motifs and invariants | Angular shear, twist, shift in the same nontrivial reciprocal profile |
| Center | Necessary and sufficient bounded-`K` conditions in the `C^2` Taylor class | Other topology, nonsmooth centers, matter cores |
| Endpoints | `A->0`, WR-L, and `A->infinity` at finite areal radius | Correlated infinite-radius and general global completions |
| Reciprocity reversal | Exact counterexample to generic full-metric `phi->-phi` symmetry | Special-profile isometries |
| Dynamics | None | Action, EOM, variation, response, evolution |
| Scale | None inserted | Physical definition of micro, `X_max`, mass |
| Compute | CPU symbolic and independent exact-rational tensors | ODE/PDE, relaxation, GPU |
