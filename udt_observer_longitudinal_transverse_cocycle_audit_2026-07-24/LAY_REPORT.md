# Lay report

We found the first mathematically honest way to carry the clock part and the
angular part of the metric together.

The simple angular-size number is not enough. It is like trying to predict
where a car will go while recording only its position and throwing away its
direction. That reduced record cannot be joined cleanly from one stretch of
road to the next.

When we keep both the transverse displacement and the direction in which it
is changing, the metric gives a complete transport rule. That rule joins
perfectly across successive path segments, reverses cleanly, and survives
even where a focused image collapses to a point.

The clock comparison also joins cleanly along the same path. So we can place
the clock rule and the complete angular rule side by side in one exact
path-to-path bookkeeping object.

That is real progress, but it is not yet the final UDT observer law. The
metric has shown that the two pieces can travel together. It has not yet
shown that they are one inseparable object, nor that the clock comparison is
always the same thing as UDT's founding reciprocal distance-depth.

The complete round branch carries the angular rule globally but has no
nontrivial clock dilation. The WR-L branch carries both locally and gives
the familiar `exp(phi)` clock behavior, but it is still centered and not a
complete observer-symmetric universe. No branch yet supplies both virtues at
once.
