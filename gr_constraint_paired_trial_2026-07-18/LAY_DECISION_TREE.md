# Lay decision tree — trying GR constraints without adopting GR wholesale

Start with one proposed constraint, not a GR bundle.

```text
Does UDT already derive it?
|
+-- YES -> label DERIVED and state its exact scope.
|
+-- NO --> Does it contradict Reciprocity, Common-Scale Neutrality,
           or the finite mirrored cell?
           |
           +-- YES -> REJECT or MODIFY it.
           |
           +-- NO --> Keep it as an explicit TRIAL only.
                     |
                     +-- Does it secretly bring companions such as EH,
                     |   metric-only fields, second order, locality,
                     |   spatial infinity, or a matter model?
                     |   |
                     |   +-- YES -> split those companions into separate trials.
                     |   +-- NO  -> run its selector/falsifier tests.
                     |
                     +-- Does it uniquely exclude the alternatives?
                         |
                         +-- YES -> selection is bounded to the proven premises.
                         +-- NO  -> retain or reject the trial, but leave the
                                   underlying UDT choice unresolved.
```

```text
Why no winner?

Next genuinely missing selector:

```
