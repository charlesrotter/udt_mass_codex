## Work pattern (when we dig a vein)

1. **Name the GR object** (paper/standard construction) — method or result?  
2. **State the transform:** simple metric + \(\phi\) positional; what is held / freed.  
3. **Re-derive natively** (CAS) — do not paste final GR formula.  
4. **OBSERVE** solution space / EL / constraints — what emerges?  
5. **Tag:** DERIVED / still GR-form conditional / empty / contradicts R1 / …  
6. **No SNe selection** on the output.

Success of a mine pass = **structure emerged or a clean negative** (this vein produces nothing new under dilation).  
Failure mode = “looks like GR so we ship it as UDT.”
