# Lay report

We learned two things, and one of them corrects the question itself.

First, “list every local rule the metric could make” is not a finite job until UDT tells us what
kind of answer the rule must return. Even with no derivatives, any smooth function of `phi` is a
different scalar rule. Add vectors, planes, densities, and higher derivatives and the list is
unbounded. The missing information is the *job description*: should the rule choose a scale, a
direction, a whole metric, a boundary value, or something else?

There is one exact conclusion. A rule that respects Common-Scale Neutrality cannot choose one scale
representative from the neutral family. It gives the same answer to every member by definition.
Physical scale selection must therefore occur when some scale-bearing global, boundary, or matter
datum enters—not through a scale-neutral local formula pretending to choose a ruler.

Second, we tested whether curvature quietly preserves a preferred direction or plane throughout
the registered metric families. The result is a stark two-way split:

- Whenever any of the registered metric sectors is active, curvature already mixes all four
  directions so completely that no proper common direction or plane survives.
- In the empty and `phi`-only sectors, the metric is exactly constant and flat. Every direction is
  allowed, so again no direction is selected.

In orchestra language, the metric instruments play a full four-dimensional chord as soon as any
metric section is active. When only `phi` plays in this particular atlas, the metric orchestra is
silent; `phi` was varied independently and does not feed back into the metric.

That last point is the useful clue. The atlas can describe metric variation and `phi` variation,
but it still lacks the native sentence saying how they must realize one another in an actual
universe. This is not permission to invent a coupling. It identifies the next source question:
does Reciprocity, CSN, the finite cell, seal, or bootstrap already contain such a relation in a form
we have not made operational?

The result does not invalidate the conditional reciprocal-toric/Hopf-compatible branch. That branch
uses separately supplied global structure and remains a control outside this 6,144-row local
ensemble. Nor does this audit determine an action, matter, mass, scale, boundary charge, or time.
