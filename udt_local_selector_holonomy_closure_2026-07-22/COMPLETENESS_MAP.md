# Completeness map

This package is one local-geometry tile.

| criterion | coverage | dropped scope |
|---|---|---|
| fields | metric and conditional scalar `phi`; coframe only through gauge audit | independent matter, source, auxiliary fields |
| action terms | none | complete bulk and boundary functional |
| full equations | none | realization relation and every Euler equation |
| domain/charts | 6,144 registered analytic local configurations | arbitrary metrics and complete global charts |
| boundary/regularity | none used | finite-cell seal variation, seams, caps, flux |
| topology | none selected | every retained global completion |
| dynamical character | local static data only | stationary, periodic, and physical-time systems |
| branches | all registered masks/carriers/points retained | field-equation solution branches |
| stability | not asked | all stability questions |
| regime | local order-zero-to-two selector types and local curvature persistence | higher jets, global holonomy, nonlocal and variational selectors |

The computation characterizes rather than filters. Flat, degenerate, ambiguous, and irreducible
rows are retained. No row is accepted or rejected for resembling matter or a desired universe.
