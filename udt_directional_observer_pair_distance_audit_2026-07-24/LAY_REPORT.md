# Lay report

We have now put the “no center” rule into the mathematics correctly.

`X_max` is not the distance from a center to an edge. It is the greatest
distance the complete metric permits between any two observers.

In the round mathematical example, every observer has an antipodal observer
exactly `pi b` away. It does not matter where either observer is or which
direction they look. There is no center and no directional variation.

We also found an important control that had to be retained. The repository
contains a squashed version of the same closed geometry. It is not an
accepted physical solution, but it demonstrates something conceptually
important:

- every location can still be equivalent;
- there is still no center;
- yet traveling around the Hopf-fiber direction and around a horizontal
  direction gives different total lengths.

So your suggestion is mathematically possible: angular effects could create
small directional variations without introducing a central observer.

We have not calculated the exact variation band yet. Doing that requires
finding where all geodesics stop being the shortest route. The squashed
example is also off shell and has no UDT clock dilation, so it is not yet an
`X_max` or CMB prediction.

The physical problem remains to put the global pair geometry and clock
dilation in the same complete metric branch.
