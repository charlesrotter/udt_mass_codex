# Lay report

The important finding is simple:

> Reciprocity gave UDT its opposite clock-and-ruler changes. It did not prove
> that their common physical size was meaningless.

The earlier mathematics showed that if one side changes by a factor, the other
changes inversely. Later, a separate decision declared that multiplying both
sides by the same local factor was only recalibration. That decision became
CSN.

The repository is unusually clear about this in hindsight. The original CSN
document calls it a postulate and says it **declares** the common factor
calibrational. The cold derivation arms were then explicitly forbidden from
questioning it. Their agreement therefore shows what follows if CSN is
accepted, not that CSN itself was discovered by the metric.

Measured `c` does not completely settle the issue. If every local clock and
ruler is multiplied by the same factor, their ratio still gives the same
`c`. So `c` alone cannot tell us the absolute size of the universe.

But that does not make the common factor unreal. Proper lengths, clock
intervals, volumes, and curvature generally change. Calling those changes
physically equivalent requires a separate principle.

This leaves two honest pictures:

1. The reciprocal metric is already the physical metric. UDT has ordinary
   freedom to choose units, but local common scaling is generally physical.
2. Only the conformal shape is fundamental, and UDT must still derive how one
   physical metric is selected.

We currently have no derivation choosing picture 2, and no selector completing
it. Therefore strong CSN is now challenged.

This does not erase the accumulated work. Reciprocal kinematics, angular
shape, topology, the conditional Hopf structure, the WR-L/SNe readout, and the
finite-box Hopfion evidence largely survive. The largest casualty is the
claim that conformal `C^2`/Bach dynamics has special physical priority.

EH is not automatically correct. The action remains open. But we no longer
need to assume that UDT must begin with a scale-free physical geometry.
