# Lay Decision Tree — Did the Metric Pick One Light Direction?

## Short answer

No—not yet. We tested the two most direct ways the metric might have done it, and both need an extra
selection law.

## What was tested

Think of `phi` as the field that tells clocks and the parallel spatial direction how much to dilate.
One hoped that the direction in which `phi` changes might itself be a light direction.

- In an ordinary static region, that direction is spatial, not lightlike.
- If `phi` changes with time, it *can* become lightlike, but only when its time and space changes are
  balanced by an additional wave-like equation. The reciprocal metric allows that balance; it does
  not command it.

The other hope was that the bending of spacetime might mark one especially repeated light direction.
Here the angular geometry matters:

- round/common-warp angular geometry marks two directions (inward and outward), or none;
- a permitted anisotropic angular geometry marks four ordinary, non-repeated directions;
- specially tuned geometries can mark one repeated direction, but nothing currently forces the
  tuning.

## Decision tree

```text
Does static dphi give a nonzero light direction?
  No -> any nonzero static gradient is spatial.
       If the gradient vanishes, there is no direction to select.

Does time-live dphi help?
  Yes, conditionally -> it can be lightlike if an extra eikonal equation holds.
  But -> Reciprocity alone does not select that equation or either sign branch.

Does Weyl curvature pick one direction?
  Round common warp -> two repeated directions, or none.
  Allowed anisotropic warp -> four simple directions.
  Petrov II/III/N -> one repeated direction, but only if a new law forces that class.

Does the present bootstrap choose among them?
  No -> it states a global consistency requirement but supplies no ranking operator yet.

Current conclusion
  The metric has narrowed the search, but has not selected one global physical null line.
```

## Why this is progress

This was not another random formula hunt. The preceding audit left exactly two local candidates:
null `d phi` and a uniquely repeated curvature direction. We have now learned where each fails and
where either could still work.

The remaining conceptual seam is a law tying the angular curvature to the reciprocal `phi` sector.
That law would have to arise from UDT rather than be borrowed from GR. It must decide which metric
families are physically admissible. Until then, the exact alternatives are evidence that the
kinematics alone are insufficient—not evidence that the larger UDT idea is dead.
