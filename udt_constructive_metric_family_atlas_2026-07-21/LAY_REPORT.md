# Lay readout

We stopped asking the metric leading questions and simply turned everything on in a controlled set
of examples.

Each constructed metric contained all ten parts: clock/depth terms, the full two-direction angular
shape, and all four ways those sectors can lean or twist into one another. Every part depended on
time, depth, and both angular coordinates.

There is an important limitation. We did not turn ten separate knobs independently. Within each of
four families, one shared knob moved all ten parts together while the four coordinates supplied four
more directions. The scan therefore explored correlated paths of at most five dimensions through a
ten-dimensional metric chart.

The plain result was remarkably consistent. The undeformed reference metrics were flat. As soon as
we moved away from those references, every one of the 128 sampled configurations developed the
full available curvature rank, angular shear, twist, and mixed depth–angular curvature.

That does not mean nature chooses those configurations, or that this behavior is mathematically
generic. It means only that coupling appeared throughout all four correlated paths once their shared
deformation knob was nonzero.

We kept `phi` as a separate field from the metric, but its sampled coefficients used the same shared
deformation knob. Its gradient was sometimes timelike and
usually spacelike; the only zero/null-class records had an exactly zero gradient at the undeformed
references. Nothing forced a nonzero lightlike `phi` gradient in this sample.

This still is not a solution of UDT's missing action problem. We constructed geometries and measured
them; we did not determine which geometries obey the universe's law. The useful progress is that we
are examining all ten local metric slots along four explicitly correlated paths, rather than a
slot-suppressed ansatz.
