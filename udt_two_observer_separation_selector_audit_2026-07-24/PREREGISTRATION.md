## Whole question

Reciprocity, Common-Scale Neutrality, finite-cell, seal, co-presence, and
bootstrap structure define a native nonnegative separation between two
observers, including:

1. the admissible observer histories;
2. which event on one history is paired with which event on the other;
3. a symmetric nonnegative separation which vanishes only at observer
   coincidence;
4. the complete angular/path dependence;
5. chart/coframe invariance and global descent across the registered
   completion and `dphi` causal classes?

This is not a search for an edge, signal mechanism, preferred answer, or
familiar external construction.

## Certification and falsification

- `DERIVED_UNIVERSAL` requires one candidate to pass all twelve gates using
  only registered UDT structure.
- `UNIQUE_CONDITIONAL` requires exact uniqueness inside an explicitly stated
  branch and may not be generalized.
- `OPEN_SELECTOR` is required if multiple candidates survive conditionally,
  all complete candidates require an unselected structure, or none descends
  across the full registered space.
- A scoped no-go may exclude only the frozen candidate universe under the
  current premises.  It may not exclude future complete-metric solutions or
  a new owner premise.
- Any noncoincident null pair assigned zero fails identity of
  indiscernibles for physical observer separation.
- Any construction using a spacelike slice must state what selects the slice.
- Any construction using a curve must state what selects the curve or
  extremization class.
- Any `dphi` construction must be tested separately on timelike, spacelike,
  null, zero, and causal-transition strata.
- Co-presence may not be upgraded from whole-solution membership to event
  pairing without an explicit registered rule.
