# Lay report

We found a real piece of the answer, but not one that works everywhere.

If `phi` changes in a time-like direction throughout a universe, then `phi`
can act like a universal page number.  Every possible observer moves through
those page numbers in the same order.  To compare observer A with observer B,
we compare where both are on page `phi = 10`, or page `phi = 11`, and so on.

Each page has an ordinary positive spatial geometry inherited from the full
metric.  That geometry includes the angular and twisting sectors, so the
shortest distance drawn on the page is a genuine complete spatial distance,
not just a radial coordinate.  No observer is made special.

This is the first clean metric-native candidate that supplies both:

- which moments on two observer histories should be compared; and
- how their spatial separation should be measured.

The limitation is decisive.  It works only when `dphi` is everywhere
time-like and never vanishes.  The static WR-L solution uses `phi` as a
spatial depth, so it does not belong to that class.  If `dphi` becomes
lightlike or zero, the page geometry degenerates.  The repository currently
contains no complete universe selecting the all-time-like branch or telling
us how to cross those degeneracies.

There is also still a ruler question.  The construction gives a natural
scale-free page geometry, but UDT has not yet said that this normalization is
the physical ruler used to measure \(X_{\max}\).

So the result is positive but conditional:

> UDT already contains a clean observer-separation construction on a
> globally time-like-`phi` branch.  It has not selected that branch as the
> universe or supplied an equally complete construction for the spatial,
> null, or zero-`dphi` branches.

That is why the final universal status remains `OPEN_SELECTOR`, rather than
“nothing was found.”
