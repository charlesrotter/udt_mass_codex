# Completeness map

## Continuous local coverage

The primary local object is the entire two-parameter positive
determinant-one angular metric family, not a finite grid. The audit must
express:

- every primitive-character chamber by an untruncated inequality;
- every possible tie wall by a group-covariant pair description;
- every wall endpoint/triple tie;
- all tie multiplicities;
- exact transformation and projected-connection laws.

Finite integer searches may be used only as adversarial controls of the
analytic result.

## Global coverage

Every row `FC01` through `FC12` must appear exactly once in:

- the completion input registry;
- the global transport outcome table;
- the completion coverage table.

Each outcome must separately record:

- torus-lattice existence;
- set-valued `W_min`;
- unique line availability;
- sign/orientation;
- projected connection;
- phase-section status;
- physical-selection status.

## Required evidence

- exact production algebra and machine-readable results;
- continuous chamber/wall/vertex atlas;
- shear-free analytic path;
- `GL(2,Z)` covariance controls;
- global transport table for all twelve completions;
- independent implementation without production import;
- exercised catch-proofs for all twelve falsification gates;
- source verification, premise audit, status ledger, lay report, and manifest.

## Explicit omissions

The audit does not classify all possible four-manifolds or all nontoric
angular geometries. It does not solve a matter field, select a density,
derive a time law, or determine which geometric invariant is physically
operative.
