# Independent-Verifier Anchor Freeze

Date: 2026-07-21

The primary independent replay uses exactly 384 configuration identities selected without reading
any motif result. For every one of the 6,144 immutable structural-ensemble IDs, compute
`SHA-256(configuration_id)`, sort by `(digest, configuration_id)`, and retain the first 384.

For each retained identity, the independent verifier must compute all 31 original families and both
nonlinear transforms—35,712 family classifications—before it reads the corresponding saved motif
rows. It must additionally replay every original numerical-margin identity after the primary blind
comparison, labeling that second set `POST_OUTCOME_MARGIN_ESCALATION` rather than blind coverage.

The anchor file itself is generated by `freeze_verifier_anchors.py`, committed before the verifier
implementation is run, and hash-checked by the verifier.
