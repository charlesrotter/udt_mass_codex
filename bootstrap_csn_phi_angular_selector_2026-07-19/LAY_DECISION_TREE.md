# Lay Decision Tree — Is the Missing Equation Already in Bootstrap/CSN?

Start with the simplest picture: the metric can change in three different ways relevant here.

1. Everything can be rescaled together. CSN says that common rescaling is calibration.
2. Time and the chosen spatial direction can change oppositely. That is the reciprocal `phi`
   direction.
3. The two angular directions can change relative to one another. That is angular shape or shear.

The exact calculation says these are three independent knobs.

```text
Does CSN by itself supply equations for all three knobs?
    |
    +-- No. If a chosen action is exactly CSN-invariant, that symmetry gives
        a gauge identity involving the common-scale trace.
            |
            +-- Does that identity secretly fix the reciprocal phi knob?
            |       +-- No. Exact trace-zero examples leave its equation nonzero.
            |
            +-- Does it secretly fix angular shape?
                    +-- No. Exact trace-zero examples leave angular shear nonzero.

Does the present bootstrap wording connect the remaining knobs?
    |
    +-- As an after-solution filter: no local equation is added.
    +-- As a varied global constraint: possibly, but the functional B is missing.
    +-- As a scale/representative selector: possibly, but the map Sigma is missing.

Can a very simple extra mathematical class pick lightlike dphi?
    |
    +-- Yes, conditionally. With only the metric and one weight-zero phi covector,
        if one insists on a local, first-derivative, homogeneous degree-two scalar
        equation with a nowhere-zero coefficient, the eikonal norm is the only one.
        The audited current source set has not selected that class.

Can conformal curvature speciality pick one repeated light direction?
    |
    +-- Not by itself. Type D has two repeated directions; type O has none.
        An additional branch and alignment rule are still required.

Do the conditional Bach or Einstein action routes force either target?
    |
    +-- No. One exact reciprocal Ricci-flat/Bach-flat metric has timelike dphi
        and four distinct curvature directions.
```

## Plain conclusion

We found where the hoped-for equation would have to live, and it is not hidden inside the present
words. CSN removes the universe's common measuring scale. It does not tell the reciprocal clock/space
distortion how to couple to angular shape. The bootstrap says complete physical universes must close
globally, but it has not yet supplied the mathematical mechanism that feeds that global demand back
into the local metric.

So the next conceptual target is sharply defined: derive the missing bootstrap-to-local map. Trying
more unrelated metrics without that map would test consequences, but would not close the selector.

An older pre-July canon entry names a `phi`-angular weld. This audit could not use that old equation
as affirmative physics under the July-1 firewall and did not regrade it. The conclusion therefore
applies to the frozen post-July C0/C1 plus current CSN/bootstrap sources, not every historical UDT
record.
