# Post-preregistration scale-symbol clarification

The preregistration used `ell` both in the conventional proper-length notation `d ell` and as a proposed relational resolution/separation scale. The derivation corrects that type collision:

- `dL_prop = B |dr|` denotes proper radial length on the supplied orthogonal static slice;
- `sigma` denotes operational probe/resolution/separation scale;
- a literal scale-indexed field is written `phi(x;sigma)`, with `x` suppressed only when working at one fixed event.

No identification of coordinate position `r`, proper length `L_prop`, and relational scale `sigma` is authorized.
