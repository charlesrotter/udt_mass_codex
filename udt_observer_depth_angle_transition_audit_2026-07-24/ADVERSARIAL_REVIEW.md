# Adversarial review

## Strongest affirmative derivation attempted

The strongest attempted route was:

```text
reciprocal positive ray
-> CSN quotient
-> unique anchored projective coordinate tanh(rho)
-> fractional observer transition
-> physical D=Xmax|tanh(rho)|.
```

The first three arrows close exactly. The last does not. It requires the
physical metric to satisfy

```text
h^-1(d rho,d rho)=cosh(rho)^4/Xmax^2
```

on the selected complete branch. No registered branch supplies that
identity.

## Strongest competing realization

The existing WR-L radial slice gives instead

```text
D=2X(1-exp(-rho)),
h^-1(d rho,d rho)=exp(2rho)/(2X)^2.
```

This is an exact conditional proper-distance result in that slice. It does
not establish a universal observer-pair diameter or prove that `2X` is
physical global `X_max`.

## Why “exponential” does not choose between them

Both bounded laws have exponential endpoint gaps. Their first-order Taylor
data are identical. Therefore neither the qualitative endpoint statement
nor the old linear approximation can select the physical law.

## Angular challenge

Exact round controls show non-collinear composition depends on both angular
dot and cross products. Replacing it with scalar fractional addition loses
information. Constant squashing preserves group composition while changing
directional lengths, proving that group structure alone does not fix metric
distance.

## Imported-comparison challenge

The local Lorentz algebra was used only as a comparison. Its commutators
generate rotations, but the reciprocal weighting fails the automorphism
test. Therefore the audit did not import Lorentz rapidity as reciprocal
depth.

## Gauge challenge

The complete triangular coframe product is exact in one representative, but
different independently gauge-related representatives give different
outputs. It cannot be promoted to physical observer composition without a
chosen section or native solder.

## Exercised failures

The independent verifier rejects:

- promoting `tanh` to physical distance;
- calling the WR-L profile uniquely selected;
- interchanging the two endpoint exponents;
- selecting a profile from first-order Taylor data;
- dropping angular data;
- importing a Lorentz solder;
- promoting chart coframe multiplication;
- making `c` alone select the profile;
- promoting global `X_max`;
- using negative physical distance;
- changing observer-local physics;
- confusing common component scaling with ratio scaling;
- applying scalar fractional composition non-collinearly;
- dropping a source or changing a source hash.
