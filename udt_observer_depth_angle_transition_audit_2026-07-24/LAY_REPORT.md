# Lay report

The simple part really is exponential.

UDT's reciprocal clock/ruler pair changes exactly like one side growing
exponentially while the other shrinks exponentially. If we display that
unbounded imbalance on a bounded dial running from `-1` to `+1`, the unique
natural projective dial is `tanh`. Changing observers then gives the clean
fractional rule that earlier work was circling.

But a dial is not automatically a ruler.

The metric currently permits two especially relevant bounded rulers:

1. the projective `tanh` ruler; and
2. the WR-L proper-distance ruler, which approaches its endpoint with a
   simpler exponential.

Both look linear close to ordinary conditions. Both become exponential near
their limit. They differ only when we look beyond the first approximation:
the `tanh` endpoint gap falls like an exponential squared, while the WR-L
gap falls like a single exponential.

So the new result does not say “we still know nothing.” It says exactly what
is already fixed and exactly what one metric quantity must decide. The
reciprocal response and the observer recentering rule are fixed. The missing
piece is the complete metric branch that tells us which dial corresponds to
actual separation between two observers.

The angular sector also matters. Along one direction the rule is simple.
When observer relationships point in different directions, their angular
parts combine like rotations: order and orientation matter. That is the
mathematical version of the angular “orchestra” joining the radial
reciprocal motif.

`c` has not disappeared. It remains the physical conversion between clocks
and lengths in the metric. What is not yet fixed is the finite spatial reach
that turns the dimensionless reciprocal depth into a measured distance.
