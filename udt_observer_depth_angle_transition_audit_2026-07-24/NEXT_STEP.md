# Next justified step

The next bounded metric-led question is:

> Does any already registered complete finite-cell metric branch force the
> transnormal invariant `B=h^-1(d rho,d rho)` to equal one of the audited
> profile families throughout the cell, including its angular dependence?

This should begin as an algebraic branch census, not a fit and not a
time-live/GPU solve. For every complete branch:

1. compute `B` without suppressing angular terms;
2. test whether `B` is a function of `rho` alone;
3. if so, integrate `dD/d rho=1/sqrt(B)`;
4. classify endpoint convergence and angular/path dependence;
5. compare the result to the preregistered profiles only after computing it.

If no existing complete branch is dynamically selected, the honest result
remains that the distance law is branch-conditional. GPU work becomes
justified only after a native equation and global branch are specified.
