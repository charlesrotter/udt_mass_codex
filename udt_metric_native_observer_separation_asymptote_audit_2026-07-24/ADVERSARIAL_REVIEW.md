# Adversarial review

This is an internal adversarial pass, not a fresh external-model review.

## Strongest attempted counter-derivation

The strongest affirmative shortcut is:

```text
1+z=exp(phi), phi->infinity at X_max, therefore X_max is already derived.
```

It fails because `exp(phi)` fixes clock dilation but contains no spatial
gradient or global path information. The exact profile family realizes
finite and infinite proper reaches with the same reciprocal clock block and
the same local calibration.

The strongest metric-native repair is not a new postulate. It is the
distance eikonal equation applied to the complete observer-rest metric:

```text
h^-1(dD,dD)=1.
```

For `D=F(phi)`, this gives the exact transnormal law recorded in the report.
This repair succeeds locally and branch-conditionally.

## Challenges

### Is the transnormal condition merely sufficient?

For one smooth scalar distance law `D=F(phi)` with nonzero `F'`, it is also
necessary:

```text
B=h^-1(dphi,dphi)=1/[F'(phi)]^2.
```

Thus `B` must be constant on `phi` level sets. Global cut loci or
non-smooth distance can weaken the smooth formulation, but cannot justify a
single smooth `D(phi)` where this level-set condition fails.

### Does a unit eikonal function equal geodesic distance?

Locally, away from cut loci, its gradient curves have unit speed and are
geodesic. Globally, eikonal normalization alone does not prove minimizing
behavior, reachability, uniqueness, completeness, or absence of caustics.
The report retains those gates.

### Did the complete coframe silently choose an observer?

Yes, `theta0` is a supplied clock leg. The algebra derives the rest geometry
given that leg. A bare Lorentzian metric does not select it. The reciprocal
UDT coframe may physically own it, but the complete nonspherical/time-live
assembly and pair-symmetric event relation remain open. The status is
therefore conditional.

### Does the angular countercontrol violate UDT?

It is not offered as a UDT solution. It lies in the current unconstrained
Lorentzian configuration class and is used only to refute the implication
“same clock law implies same global diameter.” A selected UDT equation or
global premise could exclude it later.

### Does finite WR-L proper reach prove global `X_max=2X`?

No. WR-L is a one-function spherical reduction, its `X` is local, and a
global diameter can differ from radial reach. The exact `2X` result remains
conditional to radial proper distance in that slice.

### Can `c_E` or `G_obs` remove the remaining ambiguity?

`c_E` calibrates clock and length units but does not select the spatial
gradient profile or angular completion. `G_obs` is not present in the
kinematic eikonal calculation. It can later test a mass/size bootstrap
relation after native mass exists.

## Ruling

The direct metric evaluator survives. Promotion to one numerical universal
`X_max` does not.
