# Next bounded scientific step

Apply the exact transnormal/asymptote criterion to every surviving complete
finite-cell branch without fitting or privileging a desired endpoint.

For each branch:

1. identify the physical reciprocal clock leg and allowed observer/event
   pairing;
2. compute the full
   `B=h_u^-1(dphi,dphi)`, retaining every angular and shift sector;
3. test whether `B` is single-valued on `phi` level sets;
4. if not, catalogue every depth-gradient path and its endpoint rather than
   forcing one `D(phi)`;
5. if yes, integrate `dphi/sqrt(B(phi))` and classify the endpoint as finite
   or infinite;
6. test global regularity, caustics, and completion;
7. compute the full pair-distance diameter; and
8. only then compare the diameter with the infinite-dilation depth reach.

This is CPU symbolic/ODE work for exact branch anchors. GPU exploration is
not justified until a branch supplies explicit global functions that
require broad numerical integration.
