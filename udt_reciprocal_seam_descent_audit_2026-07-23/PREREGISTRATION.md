## Exact algebraic controls

The deterministic implementation must verify:

```text
G_a D(phi) G_a^-1 = D(phi),
F_b D(phi) F_b^-1 = D(-phi).
```

For mirror coordinate reversal `x -> -x` and odd static profiles

```text
phi=x,
phi=x^3,
phi=x+x^3,
```

it must test the full pulled-back identity

```text
F_b D(phi(-x)) F_b^-1 = D(phi(x)),
```

including value, first-jet, and second-jet matching at the seal. The cubic
profile is the critical-point control: descent must remain valid even though
`dphi=0` at the seal.

For an arbitrary invertible angular transition `A`, test the block extension

```text
T=F_b direct_sum A,
D4(phi)=D(phi) direct_sum I2.
```

If

```text
T D4(phi) T^-1 = D4(-phi)
```

is independent of `A`, reciprocal descent does not by itself couple or
select the angular lift.
