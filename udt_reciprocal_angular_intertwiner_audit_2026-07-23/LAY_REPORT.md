# Lay report

We asked whether the metric itself contains a hidden coupling that naturally
locks its reciprocal clock/ruler structure to its angular structure.

The answer is: **there is a remarkably tight coupling available, but the
metric has not yet chosen the angular pattern needed to activate it.**

Imagine two pairs of gears.

- The clock/ruler pair is founded: when one gear stretches, the other
  shrinks by the opposite amount.
- The angular pair may turn in many mathematically allowed ways.

The two pairs can be locked together across the whole positional range only
when the angular gears also stretch and shrink with exactly those opposite
rates. If they do, the mirror at the finite-cell seal removes almost all
freedom in the coupling: only one relative size remains.

If the angular pair is an ordinary spectator, a rotation, or has only one
matching rate, it cannot be fully locked to the reciprocal pair by this
route.

This is useful because we no longer have to guess arbitrary couplings. For
any future complete metric branch, we can calculate its angular stretching
rule and ask one exact question: does it have the reciprocal `+1/-1`
pattern?

There are already exact local full-metric Lorentzian witnesses with the
matched pattern and nonzero cross-coupling. But there are also exact local
zero-coupling witnesses, and the inverse assignment works equally well. So
compatibility is not yet global selection.

The role of `c` is clear. It remains the real conversion between clock and
ruler units and appears explicitly in the metric. It does not set the
angular radius or the remaining mixing strength. Those require another
piece of the complete solution.

So the new clue is not “the soldering is solved.” It is:

> If the angular sector is found to echo the same reciprocal pattern, the
> coupling is almost completely determined already.
