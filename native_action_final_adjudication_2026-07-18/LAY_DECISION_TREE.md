## Shared-static-source test

Unique full dynamics is not logically necessary if inequivalent complete actions all reduce to the
same exact static sourced sector. That possibility is real enough to test, but it has not been
established here.

```text
Across at least two inequivalent complete actions, is there the same exact
finite-cell static equation on the same fields and variation domain?
|
+-- No / unknown --> Static source remains OPEN.
|
+-- Yes --> Does metric/carrier variation give the same source combination,
            rather than merely the same static energy?
            |
            +-- No / unknown --> Static source remains OPEN.
            |
            +-- Yes --> Do the actions induce the same differentiable finite-cell
                        boundary generator, reference, orientation, and normalization?
                        |
                        +-- No / unknown --> Gravitational charge/mass remains OPEN.
                        |
                        +-- Yes --> Is there a compatible nontrivial finite-cell solution,
                                    global closure, and a controlled weak/static limit?
                                    |
                                    +-- No / unknown --> Readout remains CONDITIONAL.
                                    |
                                    +-- Yes --> A scoped static mass theorem could be
                                                derived before full dynamical uniqueness.
```

The present record fails at the first three positive gates. The A/B completion families often share
the same three-dimensional energy, but they need not share its lapse/metric variation or boundary
charge. Therefore the route is `OPEN`: it corrects the D6 no-go, but it does not yet promote mass.

## If the conditional S2/EH branch is deliberately examined

Keep three statements separate:

1. The chosen EH plus minimal four-dimensional S2 branch gives
   `M_N^(0) = 2 E4` in its controlled weak regime.
2. A finite cell gives
   `M_N^(0) = Ecarrier + B_boundary + W_res`.
3. `M_N^(0) = Ecarrier` follows only on a separately proved sequence where both the boundary and
   residual vanish while the weak expansion stays controlled.

An exact critical point removes the residual, not automatically the boundary. The current finite-box
gap near -2.7 percent therefore stays visible; the old 0.05 percent centered-operator match is only a
superseded failure record.
