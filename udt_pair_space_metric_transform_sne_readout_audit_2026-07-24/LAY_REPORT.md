# Lay report

We found a clean way for UDT to have a maximum distance between any two
observers without putting Earth—or anything else—at the center.

Start with an ordinary distance between two observers. The metric can pass
that distance through any of three already discovered “saturating” curves.
Each curve keeps increasing but approaches a ceiling. All three preserve the
basic rules a distance must obey.

That is real progress: a center-free distance ceiling is mathematically
coherent. But it does not yet tell us which curve nature uses.

The supernova result then clarified something important. The WR-L curve that
fits the supernovae is not the ordinary distance measured along space. It is
the combined result of three parts of the same metric:

1. how clocks compare;
2. how the area of a light front grows;
3. how that area and clock change combine into observed brightness.

That complete WR-L combination matches the existing supernova data much
better than the complete projective/tanh alternative. The other geometries
cannot honestly be scored yet because they do not have all three pieces.

So the result is not “the data picked the exponential distance law.” It is:
“the data favor one complete way the metric assembles clocks, areas, and
light readout.”

The next job is to derive that complete two-observer assembly without using a
special center and without asking the data to invent the missing rule.
