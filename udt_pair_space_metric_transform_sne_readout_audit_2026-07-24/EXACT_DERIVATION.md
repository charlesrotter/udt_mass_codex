# Exact pair-space derivation

## 1. Conditional metric-transform theorem

Let `(M,d)` be a metric space. Let `X>0`, `kappa>0`, and let
`f:[0,infinity)->[0,1)` obey:

- `f(0)=0`;
- `f(s)>0` for `s>0`;
- `f` is increasing;
- `f` is concave.

Define

```text
D_f(p,q) = X f(kappa d(p,q)).
```

Then `D_f` is a metric.

Nonnegativity, symmetry, and identity of indiscernibles follow immediately
from those properties of `d`, positivity of `X,kappa`, and strict positivity
of `f` away from zero.

For `x,y>=0`, concavity and `f(0)=0` give

```text
f(x) >= x/(x+y) f(x+y),
f(y) >= y/(x+y) f(x+y).
```

Adding yields subadditivity:

```text
f(x+y) <= f(x)+f(y).
```

Consequently,

```text
d(p,q) <= d(p,r)+d(r,q)

=> f(kappa d(p,q))
   <= f(kappa[d(p,r)+d(r,q)])
   <= f(kappa d(p,r))+f(kappa d(r,q)).
```

This is the triangle inequality for `D_f`.

This theorem is mathematical and conditional. It does not derive the base
observer metric, event pairing, `X`, `kappa`, or the physical profile.

## 2. The three registered transforms

The registered normalized profiles are

```text
f_T(s) = tanh(s),
f_E(s) = 1-exp(-s),
f_R(s) = (2/pi) atan(sinh(2s)).
```

Their derivatives are

```text
f_T'(s)  = sech(s)^2,
f_T''(s) = -2 sech(s)^2 tanh(s);

f_E'(s)  = exp(-s),
f_E''(s) = -exp(-s);

f_R'(s)  = (4/pi) sech(2s),
f_R''(s) = -(8/pi) sech(2s) tanh(2s).
```

All three vanish at zero, are strictly increasing for finite nonnegative
`s`, are strictly concave for `s>0`, and approach one from below. Each
therefore satisfies the theorem.

If the base metric has unbounded diameter, then

```text
sup D_f = X,
```

and `X` is not attained at any finite base distance. If the base diameter is
the finite supremum `Delta`, continuity gives

```text
sup D_f = X f(kappa Delta) < X.
```

The first statement is an exact way to construct an uncrossed finite
pair-distance supremum without a preferred center. It is not a derivation of
physical `X_max`, because the base metric, event pairing, profile, and scale
remain unselected.

## 3. These are not ordinary local lengths

For strictly concave `f`, `s>0`, and integer `n>1`,

```text
n f(s/n) > f(s).
```

Therefore, subdividing a base geodesic and summing transformed segment
distances gives more than the transformed endpoint distance. `D_f` is in
general a nonlocal pair comparison, not the length generated by integrating
a local line element.

This is important for the co-present interpretation: a bounded observer-pair
comparison can be mathematically coherent without being a local signal path.
It still does not establish that this is UDT's physical comparison law.

## 4. Collinear composition

When the supplied base depth is additive along an ordered collinear chain,
the normalized transformed distance `y=f(s)` composes associatively.

Projective:

```text
y12 = (y1+y2)/(1+y1 y2),
inverse depth = atanh(y).
```

WR-L proper exponential:

```text
y12 = y1+y2-y1 y2,
inverse depth = -log(1-y).
```

Round B19:

```text
a_i = tan(pi y_i/2),
a12 = a1 sqrt(1+a2^2)+a2 sqrt(1+a1^2),
y12 = (2/pi) atan(a12),
inverse depth = (1/2)asinh(tan(pi y/2)).
```

Associativity is inherited by conjugating ordinary addition with each
profile and its inverse. For arbitrary triples, this scalar rule does not
replace angular transport, and the cut locus can still make path transport
multivalued.

## 5. WR-L contains several distinct readouts

On the registered conditional WR-L branch,

```text
A = exp(-2phi) = 1-r/X.
```

The same lapse gives:

```text
clock/redshift:            1+z = exp(phi),
areal radius:              r/X = 1-exp(-2phi),
proper radial distance:    D_prop/(2X) = 1-exp(-phi).
```

These are not interchangeable. The full registered optical relation is

```text
d_L = (1+z)^2 D_A,   D_A=r,
```

so

```text
d_L/X
 = exp(2phi)[1-exp(-2phi)]
 = exp(2phi)-1
 = (1+z)^2-1
 = z(z+2).
```

The successful SNe score therefore tests the assembly

```text
WR-L clock + WR-L areal radius + registered static optics,
```

not the WR-L proper-distance profile and not a universal pair-distance
operator.

For the registered projective J1 comparator,

```text
r/X = tanh(phi),
u = 1+z = exp(phi),
d_L/X = u^2 (u^2-1)/(u^2+1).
```

This is likewise a complete conditional readout and can be scored. The B19,
FC12, temporal-phi, and bare pair-transform candidates do not yet have the
complete clock/areal/optical join needed for an SNe score.
