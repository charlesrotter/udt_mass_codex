# Lay report

We avoided repeating an audit that had already been done and learned
something more useful.

There are now three different ways the existing metric work turns the same
general reciprocal-depth idea into a bounded shape:

1. the `tanh` projective dial;
2. the simple exponential ruler inside the local WR-L metric; and
3. a third curve produced by the complete round angular geometry.

The surprising part is that the third curve and `tanh` both close in on
their limit at the same exponential rate. From far away they look like the
same mechanism. Exact measurements near the ordinary point show that their
curvature differs, so they are not the same curve—even if we change the
units used for depth.

That means “the approach becomes exponential near `X_max`” is probably
right but is not enough to tell us which physical ruler the universe uses.
Different parts of the metric can produce that same broad behavior.

The open toric metric is flexible enough to imitate all three, but only
because one of its shape functions is still free. Choosing that function
would be us drawing the desired answer into the metric, not the metric
selecting it.

So the problem has become more concrete:

- we know the reciprocal clock response;
- we know three exact bounded geometric shapes it can resemble;
- we know how to tell those shapes apart;
- but no existing complete solution combines the clock behavior, angular
  geometry, observer pairing, and maximum distance in one object.

`c` remains present as the clock-to-length anchor. The unresolved issue is
which complete geometric realization supplies the corresponding spatial
reach.
