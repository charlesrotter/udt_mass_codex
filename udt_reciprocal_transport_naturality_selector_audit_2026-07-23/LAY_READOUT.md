# Lay readout

We tested whether the rules UDT already has pick one unique way to
compare directions from place to place.

They do not.

There is a very natural candidate built directly from the metric and
the way `phi` changes. It respects UDT's scale neutrality. But there is
also an entire family of alternatives that respects the same scale
rule, the reciprocal flip, and the finite-cell seal.

The simplest surviving alternative differs by an amount proportional
to `phi squared`. At the seal, where `phi` is zero, it looks exactly
like the original candidate. Away from the seal it gives genuinely
different transport. So the boundary cannot tell the alternatives
apart.

We also checked whether the additive reciprocal rule fixes this. It
does fix how one positional comparison combines with another. It does
not say that the local field `phi` may be shifted everywhere by an
arbitrary constant as a gauge freedom. Treating those as the same rule
would have manufactured a unique answer.

There is a useful conditional result: if UDT later derives both that
local shift rule and a strict flip-invariance rule for transport, then
the natural candidate is unique within the tested family. UDT has not
yet derived either rule in that form.

In plain terms: we found a good geometric road, but the metric currently
allows several equally legal lane markings. We should not paint one of
them “the law of motion” just because it is the cleanest.

That clears the way to return to the Hopfion question carefully. The
next useful test is whether the already verified Hopf topology depends
on choosing one of these transport laws at all. If it does not, the
topological result is robust. If it does, the missing selector is
exposed rather than hidden.
